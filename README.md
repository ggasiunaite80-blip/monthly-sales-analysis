# monthly-sales-analysis
## Project Overview
A simple Python project that analyzes product sales data using **NumPy**, **Pandas**, **Matplotlib**, and **Seaborn**.  
It calculates total revenue, average sales, best and worst-selling products, and shows sales by day of the week.

---

## Features
- Calculates total and average sales  
- Finds best and worst products  
- Groups sales by weekday  
- Creates bar charts for products and weekdays  

---

## Technologies
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn
